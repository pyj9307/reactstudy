• 사람이 읽을 수 있는 텍스트 기반의 데이터 교환 표준
  • 자바스크립트 객체 표기법을 따른다
  • 사람과 기계가 모두 읽기 편하도록 고안되었다.
  • 프로그래밍 언어와 운영체제에 독립적이다.

JSON의 구조

  • 이름(key)과 값(value)의 쌍으로 이루어진다. ex) {id:'sik', name : '김영식' ,birth : 0217 }
  • 데이터의 구분은 ,(쉼표)로 나열한다.
  • 객체(object)는 {}(중괄호)로 둘러 쌓아 표현한다.
  • 배열(array)는 [](대괄호)로 둘러 쌓아 표현한다.

문법 :
{
"데이터이름" : 값1 ,
"데이터이름2" : {값2} ,
"데이터이름3" : [값3]
}
